# switch_sdl_pong
Pong For The Nintendo Switch Using SDL C++ And libnx
clone or downloaf this repository.

Build with devkitpro, needs libnx, sdl2, sdl_mixer, sdl_image.

from msys2 cd into this directory.

create romfs filder inside this directory

run make
  
copy switch_sdl_pong.nro and the assets folder to SD:/switch/switch_sdl_pong/
